# Personal Website

A personal portfolio website built with **React (JavaScript)**.  
It showcases who I am, the projects I’ve worked on and provides a simple way for people to contact me.

## Features

- **About Me** – A short overview of my background, skills, and current focus.
- **Projects** – A curated selection of my work with descriptions and links.
- **Contact** – A direct way for people to reach out to me.

## Tech Stack

- **React (JavaScript)**
- **Vite**
- **CSS**
- Hosted on **simonavatcheva.com**
